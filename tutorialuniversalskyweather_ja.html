Quick Start Guide

  1. このページでは、このアセットの使用方法について簡単に説明します。
    MobilePreviewを使用している場合は、アウトライナで 'BP_SkyColoud' を選択し、詳細パネルでモバイルのチェックボックスを有効にしてください。

    2. これまでの操作で、少なくとも現時点では MobilePreview でのレンダリングができるはずです。

    モバイル開発環境でこの操作を行っていない場合、雲は表示されません。

    プロジェクトを正しくパッケージングすると確認できます。

    したがって、スタートアップフェーズでプロジェクトをパッケージ化するか、所望の環境でデバッグを行うことが推奨されています。

  2. 操作の手順については、Use PC と Use Android を参照してください。

    そして、ここではいくつかのカスタム機能について説明します。
    このアセットでは、主に 'BP_SkyCloud' というブループリントを使用します。
    まず、World OutlinerのDetailsパネルから操作可能な変数を説明します。

    3. Date

    ゲームの開始時に、ゲーム内の年/月/日を設定します。

    開始日から日が経過すると自動的に増加します。
    これは 'BP_SkyCloud' ブループリントの TickGraph で構成されています。

    たとえば、各月を均等に30日にするか、各月の日数を調整したい場合は、コメント 'A system that sets and increments the end of a day, the end of a month, and the end of time' 内の SelectNode の 'int' 値を変更してください。

    明確に言うと、Option0 は使用されません。 Option1 は1月を表し、Option2 は2月を表します... といった具体的な設定ができます。

    Time

    • Solar Time

      この変数は、ゲームの開始時点での開始時間を表します。
      Construction Scriptに接続されていますので、夜だけのコンテンツや朝だけのコンテンツを作成する際に、太陽の適切な状態を設定することで、作業をスムーズに進めることができます。

    • Time Scale

      この変数は、ゲーム内の時間が進む速度を表します。
      値が小さいほど（小数も含む）、進行が速くなり、大きな値は進行を遅くします。
      0に設定すると時間の経過が完全に停止します。

    Settings

    • Mobile

      最初にこちらで説明された内容を参照してください。

    • Random Weather

      このチェックボックスがTrueに設定されている場合、天候はゲーム開始から1日おきにランダムに選択されます。
      ただし、これはエディタのビューポートには反映されません。
      Falseに設定されている場合、天候は以下のエディタの天候選択で指定されたものに固定されます。

    • Auto Weather Update

      天気の更新を毎日0:00に行うかどうかを設定できます。
      基本的には、上記の 'Random Weather' がTrueに設定されているときに異なる天気をランダムに選択するために使用されます。
      （同じ天気が選択される場合があります。）
      固定の天気を使用したい場合は、Falseに設定できます。

    • Editor Weather Selection

      デフォルトの天気をビューポートおよび固定シナリオで使用する天気を設定するために、いくつかの天気オプションを選択できます。
      PIE（エディタ内で再生）モードとは異なり、パーティクルは生成されません。

      また、'Clear Sky' が選択されている場合、初期設定で雲のない晴れた青空を表します。
      そのため、'Clear Sky' の間は雲が表示されません。
      さらに、プレビューレンダリングレベルの設定によっては、霧、雨、およびその他の効果の可視性が異なるか、見えにくくなる場合があります。

      使用するデバイスや仕様に基づいてこれらの設定を調整してください。

      注意

      'Editor Weather Selection' の内容を変更したい場合、雲の密度やテクスチャなどの設定を変更するには、'BP_SkyCloud' 内の 'Set Cloud System' ファンクションノードにアクセスし、各天気に対応する変数を操作してください。

Use Android

  1. 年/月/日（ゲーム内）
  2. 時間
  3. 天気アイコン（自動）
  4. クリックで切り替え可能。フィートまたはキロメートル
  5. 天気更新がTrueの場合、天気を更新します。
  6. これをクリックすると、大気圏を抜け上昇します。（主にデバッグ目的です。）
  7. このバーを右に移動すると、ゲームのスピードが遅くなります。（ただし、最も左にある位置では時間が停止します。）
  8. Shift + F1を押してマウスを解放します。
    そして再びここをクリックすることでマウスを捕まえます。
    （マウスが捕まっている間と開いている間で表示が変わります。）
  9. 遠くに移動したい場合、主にデバッグ目的で
    クリックしてポーンのスピードを上げると、ポーンは非常に速く移動します。
  10. 手動操作で上げたり下げたりできます。
  11. ゲームを終了し、ウィンドウを閉じます。
