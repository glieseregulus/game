Quick Start Guide

  1. This page will provide a brief explanation, quick start guide, and tutorial for this mod.

    2. When you enter ARK, please go to the MOD section first. Next, search for 'Ultimate Graphic Optimizer and Compressor.'
    It should come up if you search for 'Ultimate' or something similar.

  2. Install

    Please install it as shown in the image below.
    It may take a while to complete the installation.
    In addition to the main content, there are several specially optimized meshes, so please be aware that it requires some storage space.
    Depending on future updates, the storage space required may increase a bit, so please ensure you have sufficient space available.

    3. CREATE OR RESUME GAME

    Go to the Game Settings screen from CREATE OR RESUME GAME, then navigate to MOD SETTINGS → AVAILABLE MODS → ACTIVATE MOD to activate the mod.
    Once activated, you should see an 'A' mark on the mod that is selected in yellow, and the name should appear in the ACTIVE MODS section.



    Once you've reached this point, you can start the game.
    This mod includes customization of the core game system.
    Therefore, during installation, all equipment being worn and items in the inventory may be deleted.
    While I experienced this once during debugging, it's uncertain if it was due to the mod or an issue with the game itself.
    Well, the game is still in development, and so is this mod.
    Please install with caution.
    Therefore, when introducing this mod, it is advisable to deposit all possessions in a safe place and perform a respawn.
    Please be careful during installation.

    Ultimate Graphic Optimizer and Compressor
    How to use explanation



    Let's move on to the MOD explanation.
    This MOD is currently under development.
    It is designed to enhance the graphic environment of Ark and ensure smooth operation even on low-spec PCs. When using it, please press the F8 key.
    First, try pressing ALL IN. With just this, you should notice significant changes in graphics and optimization.
    For those who simply want to make things faster and have issues with their GPU, this might be all you need.
    Additionally, as a side note, since it seems that manipulating the gamma value does not work, I have set it for use with the fixed functions of F1 and F2.

    Functional explanation

    • Hide Nearby Actors

      Mainly, this feature will render objects, excluding buildings near survivors containing wood and gatherable items, invisible.
      This is an unavoidable process for graphic improvement, and these objects are not actually destroyed and disappear.
      They just become invisible.
      Large rocks and trees may seem to have disappeared at first glance, but they do exist in those locations.

      Therefore, when flying on creatures like Argentavis, be cautious as you may collide with these objects if they were present.
      Rexes walking on rocks may appear to be floating.
      This feature could be useful when searching for specific dinosaurs.

    • Hide Distant Actors

      Contrary to 'Hide Nearby Actors,' this feature renders distant objects invisible.
      Although they may occasionally be visible during loading, they will become invisible again after a while.

    • Lowest User Settings

      You can lower the quality even more than the low-quality settings available in the regular ARK user settings.
      This is primarily suitable for those who want to speed up their game performance at all costs.

    • Lowest Sky Cloud Settings

      'r.VolumetricCloud 0' and 'Fog 0' have similar behavior, but in reality, this setting may be slightly lighter as it also consolidates the invisibility of finer details, such as child elements.

    • Compression Nanite Meshes and Textures

      This is currently applied only to the red obelisk.
      The fundamental mesh vertices are reduced to about 1/20, and textures are compressed from 8K and 4K classes to 512px, minimizing the impact on appearance.
      This is a very labor-intensive process, so please be patient.
      We will continue to add this to other areas in future updates.

    • Erase All Shadow

      It behaves the same as 'r.ShadowQuality 0'.

    • FXAA and NoMSAA Anti-Aliasing

      It reverts the anti-aliasing method to the one used in the ASE era.
      However, the effect may not be significant, and it produces noticeable jagged edges, so I personally do not recommend it.
      It may be removed in future updates.

    • Compression Ocean

      The content I highly recommend in this MOD is as follows.
      It recreates the compression mesh and material for the sea, replacing them without significantly altering the appearance.
      It behaves differently from 'r.Water.SingleLayer.Reflection 0,' and even with optimization, you should be able to observe reflections and waves normally.
      Additionally, you can apply 'r.Water.SingleLayer.Reflection 0' with a command, but I believe it's become unnecessary when combined with others.
      If you really want to make it a bit lighter, feel free to use it at your discretion.

    • Other Console Commands

      You can apply console commands that may relatively lighten the load in one go.
      These commands are mainly used for Lumen and LOD-related optimizations.

    • MOD Active Key

      This is the key binding configuration to open the UI.
      To change it, click on where it is initially set as F8 and input the desired key on your keyboard.
      This setting is initialized when the game is closed and cannot be saved, so please be careful the next time you launch the game.

    • Gamma 1

      Since the regular keyboard gamma setting wasn't available, I made one as well.
      You can also change the value by clicking on the back numbers and typing.
      The other details and specifications are the same as the MOD Active Key.
