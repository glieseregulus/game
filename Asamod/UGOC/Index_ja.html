Quick Start Guide

  1. このページでは簡単にこのMODの解説やクイックスタート、チュートリアルをしていきます。

    2. ARKへ入ったらまずMODへ入ってきてください。
    次に「Ultimate Graphic Optimizer and Compressor」を検索してください。Ultimateとかで検索すれば出ると思います。

  2. インストール

    下の画像のようにインストールしてください。
    しばらく待つとインストールできます。
    本編とは別に専用に軽量化したメッシュがいくつか入ってありますので、容量が必要ですのでご注意ください。
    今後のアップデートによってはもう少し増えると思いますので十分な容量を確保しておいてください。

    3. CREATE OR RESUME GAME

    CREATE OR RESUME GAMEからゲーム設定画面へ入り、MOD SETTINGS → AVAILABLE MODS → ACTIVATE MOD でMODをアクティブにしてください。
    アクティブになると黄色く選択されているMODにAマークが付いたり、ACTIVE MODSの方に名前が入ったりするはずです。

    ここまで来たら、ゲームを開始して大丈夫です。

    このMODには、コアゲームシステムのカスタマイズが含まれています。
    したがって、インストール時に、着用されているすべての機器とインベントリ内のアイテムが削除されることがあります。
    私はデバッグ中に一度なったのですが、以降一度もなくゲーム本体のせいだった可能性もありますのでなんともいえないのですが...。
    まぁまだこのゲーム自体が開発中ですし、このMODもまだ開発中です。
    充分に気をつけてインストールしてください。
    したがってこのMODを導入する際は持ち物をすべて安全な場所に預け、リスポーンを行っていただくことが望ましいです。

    ではMOD解説に行きましょう。
    このMODは現在開発中です。
    Arkのグラフィック環境を強化し、低スペックPCでも快適に動作することを目的としたMODです。

    ご使用の際はF8キーを押してください。
    まず始めにはALL INを押してみてください。
    これだけでかなりのグラフィックス変化と軽量化がされたと思います。
    基本的にとにかく早くしたい！GPUが駄目なんだ！！という人はこれだけでいいかもしれません。

    また、補足ですが、ガンマ値の操作が効かないようなので、F1とF2の固定動作に設定して使えるようにしました。

    ～日本のユーザーの皆様へ～

    現在、日本語環境のみで発生するいくつかの不具合が確認されており、不具合も発生しております。
    そのうちの1つは、デバッグ中にUI表示入力のF8キーを押した際に、数秒から数分後にF8が効かなくなる謎のバグです。
    慎重に検討した結果、MOD 内でこのバグに対処するのはほぼ不可能であると判断しました。
    回避策として、Google キーボードをこちらからインストールして適用すると、私の開発環境ではこのバグが解消されるようです。
    ユーザーがこのソリューションを自分の側で適用できると幸いです。

    残りの詳しい機能は下のセクションで解説します。

    機能解説

    • Hide Nearby Actors

      主に木や拾得可能なものを含む、サバイバーの近くにある建築物以外のオブジェクトを非可視化します。
      これはあくまでグラフィック改善のためのやむを得ない処理ですので実際に破壊されて消える訳ではないです。
      ただ、見えなくなるだけです。
      一見消えたように見える大きな岩や木も確かにそこに存在はしています。
      したがって、アルゲンなどで飛んでいる時それらか存在した際には引っかかりますので注意してください。
      岩の上を歩いているREXは浮いているように見えます。
      探したい恐竜がいる時には便利に使えるかもしれません。

    • Hide Distant Actors

      'Hide Nearby Actors'とは逆に遠くのオブジェクトを非可視化します。
      読み込み等で時々見えることがありますが、しばらくするとまた非可視化します。

    • Lowest User Settings

      通常のARKのユーザー設定で下げられる低品質設定より更に品質を下げてことができます。
      これは主に、とにかく早くしたい人に向いています。

    • Lowest Sky Cloud Settings

      'r.VolumetricCloud 0'と'Fog 0'と似た挙動になりますが、実際には子要素などの細かいものもまとめて非可視化しているので若干こちらの設定のほうが軽いかもしれません。

    • Compression Nanite Meshes and Textures

      これは現在、赤いオベリスクにのみ適用されています。
      根本的なメッシュの頂点数を約20分の1程度まで減らし、テクスチャは8K 4K クラスから 512pxまで極力見た目を崩さずに圧縮し、差し替えています。
      こちらは非常に手間のかかる作業なため、時間を下さい。
      今後のアップデートで随時追加していきます。

    • Erase All Shadow

      'r.ShadowQuality 0'と同じ挙動です。

    • FXAA and NoMSAA Anti-Aliasing

      アンチエイリアス方式をASE時代のものに戻します。
      しかし、あまり効果が期待できず、その割にかなりジャギーが出ますので私個人的には非推奨です。
      今後のアップデートで消すかもしれません。

    • Compression Ocean

      本MODで私が最もおすすめするコンテンツがこちらです。
      海の圧縮メッシュと圧縮マテリアルを作り直して見た目をあまり変えることなく差し替えています。
      'r.Water.SingleLayer.Reflection 0'と挙動は違い、軽量化されても反射して波も普通に確認できると思います。
      また、'r.Water.SingleLayer.Reflection 0'を更にコマンドで適用できますが他と合わせて必要ないくらいになっていると思います。
      どうしても少しでも軽くしたいというのであれば任意で使っていただいてもいいと思います。

    • Other Console Commands

      コンソールコマンド系で比較的軽くなるかもしれないコマンドを一気に適用できます。主にLumen系やLOD関連に対して使われています。

    • MOD Active Key

      このUIを開くための起動キーのキーバインド設定です。
      変えるときは初期ではF8になっているところをクリックして変更したいキーボードを入力します。
      この設定はゲーム終了と同時に初期化され、保存ができませんので次回起動時に気をつけて下さい。

    • Gamma 1

      通常のキーボードGamma設定が使えなかったのでついでに作っておきました。後ろの数字をクリックして打ち込めば値も変えることができます。他はMOD Active Keyと細かいことや仕様などは同じです。

    • Gamma 2

      同じでガンマ2です。
