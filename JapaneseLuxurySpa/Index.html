Quick Start Guide

  1. Project Settings

    インストール後アセットプロジェクトを起動したら、プロジェクト設定を開いてください。

    レンダリング項目に移り、下のように設定してください。

    Support global clip plane for Planar Reflections : True
    Ray Traced Translucent Refractions : True
    Motion Blur : False

    下の画像は設定サンプルです。

    2. 当アセットではPlaner Reflection機能を使用しており
    この設定は主にシャワー台に使っている鏡の反射で必須になる設定のようですので、鏡を絶対に使わない場合以外は設定してください。

    お風呂のシステムだけの動作を使いたい場合、この設定は必要ないかもしれません。

    Planer Reflectionについて詳しい設定を見たい人はこちらの公式ページを参考にしてみてください。

  2. Game Mode Settings

    当アセットには独自の専用Controller(PC_LuxurySpa)が内包されています。

    今回はWorldSettingsで設定してあります。下の画像のようになっているかを確認し、なっていなければ同じように設定してください。

  3. Plugin

    Pluginが有効になっているかを確認してください。

    項目は三つです。
    GameplayStateTree
    StateTree
    NiagaraFluids

    この三つです。

  4. BP_ThirdpersonCharacter

    これはあくまで注意ですが、アセット移行後などでDemoフォルダーのThirdpersonTemplateを使わなかった場合
    こちらのMeshのComponentTagにcolliderが設定されているか注意してください。

    これがないとお湯に入ったときに波紋などのレンダリングがなくなります。

    5. ゲーム上での操作

    わかりやすいようにキャラクターウィジェットのHUDで簡単な操作用UIを表示してありますが
    一応簡単に解説したいと思います。
    キャラクターがバスチェアにオーバーラップした時HUDが表示されます。
    Eキーで右手にアタッチします。


    キャラクターがバスチェアーを持っているとき、Qキーで椅子を外すことができます。
    デタッチした場合、バスチェアーは物理状態になります。

    また、右手にバスチェアーがある状態で使用可なシャワーパーテションポイントとオーバーラップした時
    Eキーでシャワーを使うことができます。


    温泉にも入ることができます。
    ポイントにオーバーラップさせてEキーで温泉を楽しみます。Qキーで終わります。
